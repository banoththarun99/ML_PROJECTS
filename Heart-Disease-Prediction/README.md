# Heart Disease Prediction Using Machine Learning

## Overview

This project predicts the likelihood of heart disease using Machine Learning algorithms. It compares multiple classification models and evaluates their performance on a medical dataset.

## Problem Statement

Heart disease is one of the leading causes of death worldwide. Early prediction can help healthcare professionals take preventive measures and improve patient outcomes.

## Dataset

Dataset: Cleveland Heart Disease Dataset (UCI Machine Learning Repository)

### Features

- age
- sex
- cp
- trestbps
- chol
- fbs
- restecg
- thalach
- exang
- oldpeak
- slope
- ca
- thal

### Target

- 0 = No Disease
- 1 = Disease
